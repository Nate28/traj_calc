Cantera thermochemistry files based upon NASA 9 polynomials.  These files include transport properties and chemical reaction data extracted from various other Cantera data files.  
 
To install, paste the files into the data directory of your Cantera installation.  For a typical user running Anaconda on Windows, this directory will be: 
  C:\Users\<username>\<anaconda_installation>\envs\<environment_name>\Lib\site-packages\cantera\data\.
